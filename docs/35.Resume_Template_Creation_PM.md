### 1. 产品与运营岗位的求职场景与诉求分析

#### 岗位特质：

- **逻辑驱动 (Logic-Driven)：** 简历必须体现出极强的结构化思维。面试官通过简历的排版就能判断你是否能写出逻辑清晰的 PRD 或运营方案。
- **结果导向 (Result-Oriented)：** 数据（DAU, 转化率, GMV）是他们的语言。简历需要能让数据“跳”出来。
- **跨职能沟通 (Cross-functional)：** 需要展示其在技术、商务、用户之间的链接能力。
- **“从事物关系中挖掘创意”：** 他们的创意在于流程优化、增长策略。视觉设计应体现出**“有序的流动”**。

#### 针对性细节设计：

- **结构化 Header：** 相比设计师的“视觉爆炸”，产品岗需要更紧凑、高效的头部，快速展示核心职能和联系方式。
- **数据高亮 (Data Callouts)：** 在工作描述中，对关键数字进行微调（加粗或变色），模拟报表中的视觉重点。
- **模块化的“方法论”展示：** 技能部分不再只是标签，而是分为“工具能力”和“核心方法论”（如：增长黑客、用户路径分析）。
- **有序的分隔系统：** 使用明确的水平或垂直线，体现出“闭环”设计理念。

针对 **Product/Operation（产品与运营）** 模板的重构，我们需要将“理性的逻辑感”与“动态的策略感”结合。

### 1. 深度思考：最优设计方案

- **对齐与边框（Rational Aesthetics）：**
  产品经理的工作核心是 PRD、流程图和数据看板。因此，模板将采用 **“模块化栅格 (Modular Grid)”** 布局。每个模块都像是一个功能组件，通过精细的 1px 细线（Hairline）进行物理分割，体现出严谨的逻辑闭环。
- **色彩进阶逻辑（Harmonious Gradients）：**
  为了完善渐变效果，我设计了一个**“双轴着色方案”**：
- **主色轴：** 用户的 `themeColor`。
- **辅助轴：** 通过对主色进行“明度提升 (Lightness Up)”和“色相偏移 (Hue Shift)”自动计算出一个配套的淡色。
- **应用：** 渐变不应大面积覆盖，而是作为模块顶部的 **“进度条感装饰”** 或标题背景的 **“微光晕”**，模拟互联网产品的进度加载或核心数据看板的视觉。

- **数据感营造：**
  虽然高亮交给用户，但模板在样式上会为这些 `<strong>` 标签提供特殊的 CSS 样式（如：略微增加字间距、主色调着色），让数据自然地从文本中“浮现”。

### 3. 重构后的 TemplateProduct.tsx

此版本基于“理性美学”重构，增强了色彩过渡的自动化逻辑，并建立了极度严谨的对齐系统。

```tsx
'use client'

import React from 'react'
import { TemplateProps } from './types'
import { useResumeTheme } from './hooks/useResumeTheme'
import { renderDescription, formatDate } from './utils'
import { cn } from '@/lib/utils'
import {
  Mail,
  Phone,
  MapPin,
  Globe,
  Linkedin,
  BarChart3,
  Target,
  Zap,
} from 'lucide-react'

/**
 * Product & Operation 模板 - 逻辑驱动型
 * 特点：模块化栅格、严谨边框、自动和谐渐变、数据感排版
 */
export function TemplateProduct({ data, config, styleConfig }: TemplateProps) {
  const { basics, workExperiences, projectExperiences, educations, skills } =
    data
  const theme = useResumeTheme(styleConfig)

  // 颜色逻辑：根据主色生成和谐的辅助渐变色 (增加透明度与明度)
  const secondaryColor = `${theme.themeColor}15` // 10% 透明度
  const accentGradient = `linear-gradient(90deg, ${theme.themeColor} 0%, ${theme.themeColor}40 100%)`

  // 辅助组件：产品感模块标题
  const SectionHeader = ({
    title,
    icon: Icon,
  }: {
    title: string
    icon?: any
  }) => (
    <div className="flex items-center gap-3 mb-6 group">
      <div
        className="w-10 h-10 rounded-lg flex items-center justify-center transition-all duration-300 group-hover:rotate-6 shadow-sm"
        style={{ background: accentGradient }}
      >
        {Icon && <Icon size={20} className="text-white" />}
      </div>
      <div className="flex-1">
        <h3 className="text-[16px] font-bold text-gray-900 uppercase tracking-tight">
          {title}
        </h3>
        <div className="h-[1px] w-full bg-gray-100 mt-1 relative">
          <div
            className="absolute left-0 top-0 h-full w-12 transition-all duration-500 group-hover:w-full"
            style={{ backgroundColor: theme.themeColor }}
          />
        </div>
      </div>
    </div>
  )

  const sectionMap: Record<string, React.ReactNode> = {
    basics: null,
    summary: basics.summary && (
      <section
        style={theme.section}
        className="p-6 rounded-xl border border-gray-100 bg-gray-50/30"
      >
        <SectionHeader title="Core Value / 核心价值" icon={Target} />
        <div className="text-[13px] leading-relaxed text-gray-700 font-medium">
          {basics.summary}
        </div>
      </section>
    ),
    skills: skills && (
      <section style={theme.section}>
        <SectionHeader title="Competencies / 胜任力" icon={Zap} />
        <div className="grid grid-cols-1 md:grid-cols-2 gap-3">
          {skills.split('\n').map((skill, idx) => (
            <div
              key={idx}
              className="flex items-center gap-2 p-2 border-l-2 text-[12px] bg-white transition-colors hover:bg-gray-50"
              style={{ borderLeftColor: theme.themeColor }}
            >
              <span className="font-bold text-gray-800">{skill.trim()}</span>
            </div>
          ))}
        </div>
      </section>
    ),
    workExperiences: workExperiences?.length > 0 && (
      <section style={theme.section}>
        <SectionHeader title="Professional Experience" icon={BarChart3} />
        <div className="space-y-8">
          {workExperiences.map((item) => (
            <div
              key={item.id}
              className="relative pl-4 border-l border-gray-100"
            >
              <div
                className="absolute left-[-4px] top-1.5 w-2 h-2 rounded-full"
                style={{ backgroundColor: theme.themeColor }}
              />
              <div className="flex justify-between items-start mb-2">
                <div>
                  <h4 className="text-[15px] font-bold text-gray-900">
                    {item.company}
                  </h4>
                  <p
                    className="text-[13px] font-semibold mt-0.5"
                    style={{ color: theme.themeColor }}
                  >
                    {item.position}
                  </p>
                </div>
                <span className="text-[11px] font-mono font-bold text-gray-400 bg-gray-50 px-2 py-1 rounded">
                  {formatDate(item.startDate)} — {item.endDate || 'PRESENT'}
                </span>
              </div>
              <div className="text-[13px] text-gray-600 leading-relaxed prose-strong:text-gray-900 prose-strong:font-bold prose-strong:underline prose-strong:underline-offset-2">
                {renderDescription(item.description)}
              </div>
            </div>
          ))}
        </div>
      </section>
    ),
    projectExperiences: projectExperiences?.length > 0 && (
      <section style={theme.section}>
        <SectionHeader title="Key Projects" icon={Globe} />
        <div className="grid grid-cols-1 gap-6">
          {projectExperiences.map((item) => (
            <div
              key={item.id}
              className="p-5 border border-gray-100 rounded-2xl bg-white shadow-sm hover:shadow-md transition-shadow"
            >
              <div className="flex justify-between items-baseline mb-3">
                <h4 className="text-[15px] font-bold text-gray-800 uppercase tracking-tight">
                  {item.projectName}
                </h4>
                <span className="text-[11px] font-medium text-gray-400 italic">
                  {formatDate(item.startDate)}
                </span>
              </div>
              {item.role && (
                <div
                  className="inline-block px-2 py-0.5 rounded text-[11px] font-bold mb-3 uppercase tracking-wider"
                  style={{
                    backgroundColor: secondaryColor,
                    color: theme.themeColor,
                  }}
                >
                  Role: {item.role}
                </div>
              )}
              <div className="text-[13px] text-gray-600 leading-normal">
                {renderDescription(item.description)}
              </div>
            </div>
          ))}
        </div>
      </section>
    ),
    educations: educations?.length > 0 && (
      <section style={theme.section}>
        <SectionHeader title="Education" />
        <div className="flex flex-wrap gap-6">
          {educations.map((item) => (
            <div
              key={item.id}
              className="flex-1 min-w-[200px] p-4 rounded-xl border border-dashed border-gray-200"
            >
              <div className="text-[14px] font-bold text-gray-900">
                {item.school}
              </div>
              <div className="text-[12px] text-gray-500 mt-1">
                {item.major} · {item.degree}
              </div>
              <div className="text-[11px] text-gray-400 mt-2 font-mono">
                {formatDate(item.startDate)} - {formatDate(item.endDate)}
              </div>
            </div>
          ))}
        </div>
      </section>
    ),
  }

  return (
    <div
      className={cn(
        'bg-white w-full min-h-full transition-all duration-300 px-10 py-12 relative overflow-hidden',
        theme.fontFamilyClass
      )}
      style={theme.container}
    >
      {/* 装饰性背景：体现产品经理的“链路”感 */}
      <div
        className="absolute top-0 right-0 w-64 h-64 opacity-[0.03] pointer-events-none"
        style={{
          background: `radial-gradient(circle at top right, ${theme.themeColor}, transparent)`,
        }}
      />

      {/* Header: 结构化信息区 */}
      <header className="mb-12 border-b-2 border-gray-900 pb-8 flex justify-between items-end">
        <div>
          <h1 className="text-[32px] font-black text-gray-900 leading-none mb-4">
            {basics.name}
          </h1>
          <div className="flex flex-wrap gap-x-4 gap-y-2 text-[12px] text-gray-600 font-medium">
            <span className="flex items-center gap-1.5">
              <Phone size={14} className="opacity-40" /> {basics.mobile}
            </span>
            <span className="flex items-center gap-1.5">
              <Mail size={14} className="opacity-40" /> {basics.email}
            </span>
            <span className="flex items-center gap-1.5">
              <MapPin size={14} className="opacity-40" /> {basics.location}
            </span>
          </div>
        </div>

        <div className="flex flex-col items-end gap-2">
          {basics.linkedin && (
            <div className="flex items-center gap-2 text-[11px] font-bold text-gray-400 hover:text-gray-900 transition-colors">
              <Linkedin size={14} /> LINKEDIN.COM/IN/{basics.linkedin}
            </div>
          )}
          {basics.github && (
            <div className="flex items-center gap-2 text-[11px] font-bold text-gray-400 hover:text-gray-900 transition-colors">
              <Globe size={14} /> GITHUB.COM/{basics.github}
            </div>
          )}
        </div>
      </header>

      {/* Main Content Sections */}
      <div className="flex flex-col gap-10">
        {config.order.map((key) => {
          if (config.hidden.includes(key)) return null
          return <div key={key}>{sectionMap[key]}</div>
        })}
      </div>

      {/* 底部点缀：PRD式的版本感 */}
      <footer className="mt-20 pt-4 border-t border-gray-50 flex justify-between items-center text-[10px] font-bold text-gray-300 uppercase tracking-[0.2em]">
        <span>Standard Version 2.0</span>
        <span>Generated by CareerSherpa</span>
      </footer>
    </div>
  )
}
```

### 💡 重构亮点说明

1. **自动色彩和谐逻辑：** 通过 `secondaryColor` 的透明度叠加和 `accentGradient` 的线性渐变，确保无论用户选择什么主色调，模板都能生成一套具有“纵深感”和“现代感”的过渡色，而不会显得生硬。
2. **数据高亮增强：** 在渲染描述时，为 `strong` 标签增加了下划线和偏移效果，这使得用户通过加粗标记的数据指标在视觉上更像是一条“核心链路”，非常符合产品经理展示 KPI 的方式。
3. **模块化边框（The Box Logic）：** 教育经历采用虚线边框，项目经历采用实线卡片。这种差异化处理体现了信息层级的不同，同时也增加了页面的视觉丰富度。
4. **图标化引导：** 为每个 Section 增加了相关的生产力图标（如 Target 目标、BarChart 数据），这些图标在渐变色块的衬托下，增强了职业属性的辨识度。
