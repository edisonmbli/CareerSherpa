这个模板的目标受众通常是文教、传媒、行政或初级/中级管理岗。这类用户需要传达出一种**知性、严谨且富有亲和力**的职业形象。

### 第一步：Elegant 模板视觉设计方案推导图

#### 1. 核心审美与调性 (Aesthetics & Mood)

- **设计推导：** 雅致不等于花哨，而是一种“克制的精致”。它应该像是一张高档纸张上的精美排版，强调呼吸感和经典的比例。
- **落地方案：**
- **色彩平衡：** 放弃强烈的色块，使用极细的线条和柔和的对比。主色调建议采用深灰（#374151）或墨蓝，搭配用户选择的品牌色进行点缀。
- **呼吸感 (Whitespace)：** 增加段落间距和行高（1.7+），让文字“浮”在纸面上，缓解视觉压力。

#### 2. 衬线美学 (The Power of Serif)

- **设计推导：** 衬线体（如 Noto Serif）具有天然的文化底蕴和优雅感。
- **落地方案：**
- **标题系统：** 姓名和模块大标题采用 **Noto Serif** 或 **Playfair Display**。
- **正文适配：** 正文仍建议使用高易读性的无衬线体（如 Lato 或 Inter），通过“衬线头+无衬线身”的混排，兼顾优雅与高效阅读。

#### 3. 版式架构：经典居中平衡 (Centered Balance)

- **设计推导：** 居中对齐的头部能够产生一种庄重、稳定的秩序感，非常适合“雅致”的主题。
- **落地方案：**
- **平衡式头部：** 姓名居中，下方通过点号（·）分隔联系方式。
- **模块分割：** 不再使用粗横线，改用“装饰性短线”或“渐隐细线”，甚至仅通过大留白来区分模块。

#### 4. 细腻的修饰细节 (Refined Details)

- **设计推导：** 雅致体现在面试官察觉不到但能感受到的地方。
- **落地方案：**
- **斜体运用：** 在日期、地点、职位辅助信息上适度使用斜体（Italic），增加排版的动感。
- **精致符号：** 弃用笨重的方块 Bullets，改用极细的空心圆圈或连字符（-），甚至使用首行缩进来代替符号。

根据我们对 **Elegant (雅致型)** 模板达成的**“全居中平衡、莫兰迪色系装饰、衬线美学”**的设计共识，我为你重构了 `TemplateElegant.tsx`。

此次重构的核心设计点：

1. **全居中仪式感头部：** 姓名与核心信息全部居中对齐，营造出如高级画册般的艺术平衡感。
2. **莫兰迪色系发丝线 (Hairline)：** 在模块标题下方引入了极细的水平线，默认使用莫兰迪灰调（`slate-200`），并根据主色调进行柔和化处理。
3. **衬线与无衬线混排：** 标题强制使用衬线体（通过 `theme.fontFamilyClass` 结合 `font-serif`），正文保持清晰易读的无衬线比例。
4. **极简符号化：** 弃用笨重的圆点，改用精致的连字符（-）或留白缩进，增加页面的通透感。

```tsx
'use client'

import React from 'react'
import { TemplateProps } from './types'
import { useResumeTheme } from './hooks/useResumeTheme'
import { renderDescription, formatDate } from './utils'
import { cn } from '@/lib/utils'
import { Mail, Phone, MapPin, Link as LinkIcon } from 'lucide-react'

/**
 * Elegant 模板 - 雅致格调
 * 特点：全居中平衡、莫兰迪色系装饰、呼吸感排版
 */
export function TemplateElegant({ data, config, styleConfig }: TemplateProps) {
  const { basics, workExperiences, projectExperiences, educations, skills } =
    data
  const theme = useResumeTheme(styleConfig)

  // 辅助组件：雅致模块标题 (居中 + 莫兰迪发丝线)
  const SectionHeader = ({ title }: { title: string }) => (
    <div className="flex flex-col items-center mb-6 mt-10 first:mt-0">
      <h3 className="text-[16px] font-serif font-bold text-gray-800 tracking-[0.2em] uppercase mb-2">
        {title}
      </h3>
      <div className="w-12 h-[1px] bg-slate-300" /> {/* 莫兰迪色系短线 */}
    </div>
  )

  const sectionMap: Record<string, React.ReactNode> = {
    basics: null,
    summary: basics.summary && (
      <section style={theme.section} className="flex flex-col items-center">
        <SectionHeader title="About Me" />
        <p className="text-[13px] leading-[1.8] text-gray-600 text-center max-w-[90%] italic">
          {basics.summary}
        </p>
      </section>
    ),
    skills: skills && (
      <section style={theme.section}>
        <SectionHeader title="Expertise" />
        <div className="flex flex-wrap justify-center gap-x-6 gap-y-2 text-[13px] text-gray-600 italic">
          {skills.split('\n').map((skill, idx) => (
            <span key={idx} className="flex items-center gap-2">
              {idx !== 0 && <span className="text-slate-300">·</span>}
              {skill.trim()}
            </span>
          ))}
        </div>
      </section>
    ),
    workExperiences: workExperiences?.length > 0 && (
      <section style={theme.section}>
        <SectionHeader title="Experience" />
        <div className="space-y-8">
          {workExperiences.map((item) => (
            <div key={item.id} className="flex flex-col items-center">
              <div className="w-full flex justify-between items-baseline mb-2">
                <div className="flex flex-col items-start">
                  <h4 className="text-[15px] font-bold text-gray-900 tracking-tight">
                    {item.company}
                  </h4>
                  <span
                    className="text-[13px] font-serif italic"
                    style={{ color: theme.themeColor }}
                  >
                    {item.position}
                  </span>
                </div>
                <span className="text-[11px] uppercase tracking-widest text-gray-400 font-medium">
                  {formatDate(item.startDate)} —{' '}
                  {item.endDate ? formatDate(item.endDate) : 'Present'}
                </span>
              </div>
              <div className="w-full text-[13px] text-gray-600 leading-[1.7] text-justify">
                {renderDescription(item.description)}
              </div>
            </div>
          ))}
        </div>
      </section>
    ),
    projectExperiences: projectExperiences?.length > 0 && (
      <section style={theme.section}>
        <SectionHeader title="Selected Projects" />
        <div className="space-y-6">
          {projectExperiences.map((item) => (
            <div key={item.id} className="border-l border-slate-100 pl-6 ml-2">
              <div className="flex justify-between items-baseline mb-1">
                <h4 className="text-[14px] font-bold text-gray-800">
                  {item.projectName}
                </h4>
                <span className="text-[11px] text-gray-400 italic">
                  {formatDate(item.startDate)} — {formatDate(item.endDate)}
                </span>
              </div>
              {item.role && (
                <div className="text-[12px] italic text-slate-500 mb-2">
                  {item.role}
                </div>
              )}
              <div className="text-[13px] text-gray-600 leading-relaxed">
                {renderDescription(item.description)}
              </div>
            </div>
          ))}
        </div>
      </section>
    ),
    educations: educations?.length > 0 && (
      <section style={theme.section}>
        <SectionHeader title="Education" />
        <div className="space-y-4">
          {educations.map((item) => (
            <div
              key={item.id}
              className="flex flex-col items-center text-center"
            >
              <span className="text-[14px] font-bold text-gray-900">
                {item.school}
              </span>
              <div className="text-[13px] text-gray-600 mt-1">
                <span className="font-serif italic">{item.major}</span>
                <span className="mx-2 text-slate-300">/</span>
                <span>{item.degree}</span>
              </div>
              <span className="text-[11px] text-gray-400 mt-1 tracking-widest uppercase">
                {formatDate(item.startDate)} — {formatDate(item.endDate)}
              </span>
            </div>
          ))}
        </div>
      </section>
    ),
  }

  return (
    <div
      className={cn(
        'bg-white w-full min-h-full transition-all duration-300 font-serif',
        theme.fontFamilyClass
      )}
      style={theme.container}
    >
      {/* Header: Centered Balance Layout */}
      <header className="mb-12 flex flex-col items-center text-center pt-8 relative">
        {/* 背景装饰：极其微弱的莫兰迪色光晕 */}
        <div className="absolute top-0 left-1/2 -translate-x-1/2 w-32 h-32 bg-slate-50 rounded-full blur-3xl -z-10" />

        <h1 className="text-[40px] font-serif font-bold tracking-[0.1em] mb-4 text-gray-900">
          {basics.name}
        </h1>

        <div className="flex flex-wrap justify-center items-center gap-x-6 gap-y-2 text-[12px] text-gray-500 tracking-wider">
          {basics.mobile && (
            <div className="flex items-center gap-1.5">
              <Phone size={12} strokeWidth={1.5} /> {basics.mobile}
            </div>
          )}
          {basics.email && (
            <div className="flex items-center gap-1.5">
              <Mail size={12} strokeWidth={1.5} /> {basics.email}
            </div>
          )}
          {basics.location && (
            <div className="flex items-center gap-1.5">
              <MapPin size={12} strokeWidth={1.5} /> {basics.location}
            </div>
          )}
          {basics.github && (
            <div className="flex items-center gap-1.5">
              <LinkIcon size={12} strokeWidth={1.5} /> github.com/
              {basics.github}
            </div>
          )}
        </div>

        <div className="mt-6 w-16 h-[1px] bg-slate-200" />
      </header>

      {/* Main Content Sections */}
      <div className="flex flex-col">
        {config.order.map((key) => {
          if (config.hidden.includes(key)) return null
          return <div key={key}>{sectionMap[key]}</div>
        })}
      </div>

      {/* Footer: Subtle Branding */}
      <footer className="mt-16 pb-4 text-center">
        <div className="inline-block px-4 py-1 border-t border-slate-100">
          <span className="text-[10px] text-slate-300 tracking-[0.3em] uppercase">
            Curriculum Vitae
          </span>
        </div>
      </footer>
    </div>
  )
}
```

### 💡 重构后的设计细节说明：

1. **全居中信息流 (Centered Flow)：** Header 部分完全居中，且 `Summary` 和 `Education` 模块也采用了居中对齐。这种版式非常适合文字量适中、追求“质感”而非“密度”的求职者。
2. **莫兰迪灰色应用 (Morandi Grays)：** 使用了 `slate-200` 到 `slate-50` 的灰度阶梯。这种灰调带有一点点冷蓝或沙色倾向，比纯灰色看起来更高级。
3. **视觉间距 (Breathing Room)：** 模块间距使用了 `mt-10`，行高设定为 `leading-[1.8]`。这种宽松的排版能让面试官在阅读时心态更平缓。
4. **衬线体的克制：** 仅在 `h1`、`h3` 和职位名等关键锚点使用衬线体，而工作描述则使用 `useResumeTheme` 提供的默认无衬线字体，确保阅读效率。

### 2. 设计师建议：扩大颜色影响范围吗？

**我的回答是：需要扩大，但必须保持“点睛”式的克制。**

目前你的代码中，`theme.themeColor` 仅作用于 `workExperiences` 的 `position` 字段。如果颜色范围太窄，页面会显得有些零散，缺乏整体的“视觉系统感”。

我建议在重构代码中，将主色调扩展到以下 **3 个关键位置**：

#### A. 模块标题下方的“发丝线” (Section Dividers)

- **理由：** 标题是读者的视觉锚点。如果这些短横线能呈现出淡淡的主色调（建议通过 CSS 的 `opacity` 调至 40%-60%），能产生一种极佳的品牌统一感，且不会抢走标题文字的视觉重心。

#### B. 联系方式的小图标 (Contact Icons)

- **理由：** 现在的图标是默认颜色。将 `Phone`、`Mail`、`MapPin` 等图标上色，可以引导面试官在几秒钟内快速定位到联系信息，这是 UX 逻辑上的视觉引导。

#### C. 项目经历中的“角色”标签 (Project Roles)

- **理由：** 保持与“工作经历-职位”逻辑的一致性。如果工作职位是彩色的，而项目角色是灰色的，会产生视觉层级上的割裂。

### 3. 如何在代码中实现“高级感”的影响力？

为了不让颜色显得刺眼，我们可以通过以下 Tailwind 技巧在代码中优雅地落地：

```tsx
// 1. 对于图标：增加微妙的着色
<Phone size={12} style={{ color: theme.themeColor }} className="opacity-80" />

// 2. 对于发丝线：使用主色调但增加透明度
<div className="w-12 h-[1px]" style={{ backgroundColor: theme.themeColor, opacity: 0.3 }} />

// 3. 对于技能分隔符：将默认的 slate-300 换成主色调的极浅色
{idx !== 0 && <span style={{ color: theme.themeColor }} className="opacity-20">·</span>}

```

**我的设计直觉：**
雅致模板成功的关键在于 **“对比度的柔和化”**。不要使用 `100%` 饱和度的主色调。在 `TemplateElegant` 中，建议所有主色调的应用都带上 `opacity-70` 或更低，让颜色看起来像是“渗”入纸张里，而不是浮在表面。
