# Landing Page Assets Guide

To maintain a high-quality, professional aesthetic for the CareerShaper landing page, please follow these guidelines when creating or sourcing images.

## 1. Hero Visual (Placeholder: `HeroSection`)
**Goal:** Showcase the "Workbench" experience â€“ a powerful, all-in-one workspace for job seekers.
*   **Content:** A high-fidelity mockup of the `/workbench` interface.
*   **Composition:** A laptop or browser window frame showing the dashboard with active tasks (Match Analysis, Resume Customization).
*   **Style:** Clean, modern UI. Use slight perspective or "floating" elements to add depth.
*   **Dimensions:** 1600x900px (16:9 aspect ratio).
*   **Format:** PNG or WebP (transparent background preferred if using floating elements).
*   **Recommendation:** Use a tool like **Screen Studio** or **Figma** to create a polished mockup. Avoid generic stock photos of people shaking hands.

## 2. Feature Showcase Icons
**Goal:** Clear, distinct iconography for the main tabs (Match, Customize, Interview).
*   **Current:** Lucide React icons (`BarChart3`, `FileEdit`, `Mic2`).
*   **Optimization:** Consider custom 3D icons or illustrated spot graphics for the active state to add "pop".
*   **Style:** Consistent stroke weight and color palette (Primary Blue/Slate).

## 3. Resume Templates Preview
**Goal:** Show the variety and quality of exportable resumes.
*   **Content:** A collage or carousel of the 8 available templates (Standard, Technical, Creative, etc.).
*   **Style:** "Fanned out" cards or a grid layout.
*   **Interaction:** Hovering over a template could zoom in or show a "Use Template" button.

## 4. "How it Works" / Philosophy Diagram
**Goal:** Visualize the `Raw Experience -> Structured Assets -> Tailored Applications` flow.
*   **Content:** An abstract flow diagram.
*   **Elements:**
    *   *Input:* Chaos/Unstructured text blocks.
    *   *Process:* A funnel or "prism" representing the AI/Asset structuring.
    *   *Output:* Clean, organized documents (Resume, Analysis).
*   **Style:** Minimalist line art or isometric 3D.

## 5. Share/Social Preview (Open Graph)
**Goal:** Attractive preview when shared on social media.
*   **Content:** Brand Logo + Value Proposition ("AI Job Assistant").
*   **Dimensions:** 1200x630px.
*   **Format:** JPG/PNG.

## General Visual Guidelines
*   **Color Palette:**
    *   **Primary:** Slate/Stone (Professional, Document-like).
    *   **Accent:** Blue/Indigo (Trust, Tech).
    *   **Highlight:** Emerald (Success, High Match).
*   **Texture:** Subtle noise/grain is used in the app to mimic paper. Keep this consistent in marketing assets.
*   **Shadows:** Soft, diffused shadows (`shadow-xl`, `shadow-2xl`) to create depth, avoiding harsh black shadows.
